# Project Roadmap

This document outlines the planned evolution of the **Wiki** repository. Milestones are subject to change based on community feedback and personal priorities.

## 🎯 High-Level Vision
To become a premier open-source personal knowledge base that inspires others to build their own digital gardens while providing high-value, searchable data across multiple domains.

## 🏗️ 2026 Q2: Foundation & Structure (Current)
- [x] Migrate to Docusaurus 3.
- [x] Implement initial domain structure (Tech, Finance, Entertainment).
- [x] Create comprehensive governance and community documentation.
- [x] Implement SEO optimizations and custom metadata.
- [ ] Refine the Mind Map generator to support interactive SVG exports.

## 🚀 2026 Q3: Domain Expansion
- [ ] **Startups & Growth**: Add case studies and unit economics deep-dives.
- [ ] **Health & Wellness**: Document longevity research, biohacking notes, and mental health frameworks.
- [ ] **Learning Paths**: Create structured curricula for Software Engineering and Data Science.
- [ ] **Tooling**: Add a dedicated section for hardware and software tools used in my daily workflow.

## 🛠️ 2026 Q4: Automation & Integration
- [ ] **Automated Content Scrapers**: Integrate scripts to fetch and update market data or movie ratings.
- [ ] **Newsletter Integration**: Set up an automated weekly summary of wiki updates.
- [ ] **Search Enhancements**: Implement Algolia DocSearch for lightning-fast querying.

## 🌌 Future Explorations
- [ ] **AI-Powered Insights**: Integrate a local LLM or RAG system to query the wiki content directly.
- [ ] **Interactive Visualizations**: Add complex graphs for financial data and tech stacks.
